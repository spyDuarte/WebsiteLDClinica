# Website para Médico - Portfolio Profissional

Website profissional desenvolvido para médico recém-formado com atuação em Santa Catarina e Rio de Janeiro.

## Características

- Design moderno e responsivo
- Navegação intuitiva
- Seções completas:
  - Apresentação profissional
  - Especialidades e serviços
  - Localizações (SC e RJ)
  - Formulário de contato
- Totalmente otimizado para dispositivos móveis
- Animações suaves e interativas

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e Grid/Flexbox)
- JavaScript (ES6+)
- Google Fonts (Poppins)

## Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Personalize as informações do médico:
   - Nome e CRM nos arquivos HTML
   - Endereços dos consultórios
   - Telefones e emails de contato
   - Especialidades e serviços
   - Redes sociais

## Personalização

### Informações do Médico
Edite o arquivo `index.html` e substitua:
- "Dr. Nome do Médico" pelo nome real
- Números de CRM (CRM-SC e CRM-RJ)
- Endereços dos consultórios
- Telefones de contato
- Email de contato
- Links das redes sociais

### Cores e Estilo
Edite o arquivo `styles.css` na seção `:root` para alterar as cores:
```css
--primary-color: #2563eb;
--secondary-color: #1e40af;
--accent-color: #3b82f6;
```

## Estrutura de Arquivos

```
WebsiteLDClinica/
├── index.html      # Página principal
├── styles.css      # Estilos e design responsivo
├── script.js       # Interatividade e funcionalidades
└── README.md       # Este arquivo
```

## Funcionalidades

- Menu responsivo para dispositivos móveis
- Scroll suave entre seções
- Validação de formulário de contato
- Formatação automática de telefone (formato brasileiro)
- Notificações visuais para feedback do usuário
- Animações ao scroll

## Deploy

Para colocar o site no ar, você pode usar:
- GitHub Pages
- Netlify
- Vercel
- Ou qualquer servidor web tradicional

Basta fazer upload dos arquivos HTML, CSS e JS.
